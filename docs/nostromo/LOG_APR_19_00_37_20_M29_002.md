## What I'd tell the next agent

- **Don't gate M29 on M29_001.** User explicitly parallelized.
- **Use Zig comptime-accumulator pattern for enum-variant invariants.** `inline for { if (mem.eql) @compileError }` looks right but fires unconditionally at comptime on every iteration — the condition is comptime-known but the compile-error is eagerly evaluated. See `src/cli/commands.zig` "Subcommand enum has no run variant" test for the working form.
- **`agent_relay.zig` is now a primitive library.** Any future feature that needs a workspace-scoped LLM relay can call `innerRelay(hx, req, workspace_id, system_prompt)` directly. The four deleted symbols (two prompts, two inner wrappers) are comptime-guarded against reintroduction.
