Usage: zombiectl [options] [command]

autonomous agent platform

Options:
  -v, --version                 Show version
  --api <url>                   API base URL
  --json                        Machine-readable JSON output (default: false)
  --no-input                    Disable interactive prompts
  --no-open                     Skip auto-opening the browser on login
  -h, --help                    Show this help

Commands:
  login [options]               Authenticate via browser
  logout                        Clear stored credentials
  doctor                        Diagnose CLI configuration and connectivity
  workspace                     Manage workspaces
  agent                         Manage external agent API keys
  grant                         Manage integration grants
  tenant                        Tenant-scoped commands
  billing                       Tenant billing dashboard
  install [options]             Register a zombie from a local skill bundle
  list [options]                List zombies in the active workspace (paginated)
  status [zombie_id]            Show zombie status (workspace-wide if no id)
  stop <zombie_id>              Halt the running session (resumable)
  resume <zombie_id>            Resume from stopped or auto-paused
  kill <zombie_id>              Mark terminal (irreversible)
  delete <zombie_id>            Hard-delete a killed zombie
  logs [options] [zombie_id]    Tail zombie activity
  events [options] <zombie_id>  Page through historical events
  steer <zombie_id> <message>   Send a message; stream the response
  credential                    Workspace credential vault
  help [command]                display help for command

Subcommands:
  workspace add
  workspace list
  workspace use
  workspace show
  workspace credentials
  workspace delete
  agent add
  agent list
  agent delete
  grant list
  grant delete
  tenant provider show
  tenant provider add
  tenant provider delete
  billing show
  credential add
  credential show
  credential list
  credential delete

Environment variables:
  ZOMBIE_API_URL    API base URL (overridden by --api)
  ZOMBIE_TOKEN      Auth token (overridden by login)
  ZOMBIE_API_KEY    API key for service auth
  ZOMBIE_STATE_DIR  Directory for local CLI state files
  NO_COLOR          Any non-empty value disables color
